# 内部资金转移定价（FTP）管理办法

**发布部门**：本行计划财务部 · 资产负债管理部
**发布日期**：2025-11-01
**生效日期**：2025-12-01
**版本**：v4.0

---

## 第一章 总则

**第一条** 为规范本行内部资金转移定价（Funds Transfer Pricing，FTP）管理，引导业务结构优化，准确核算分支机构和产品的经营效益，依据《商业银行内部资金转移定价指引》及本行《资产负债管理办法》制定本办法。

**第二条** **FTP** 是总行司库与各业务部门、分支机构之间，按照特定规则对每笔资金的来源（负债）和运用（资产）进行内部计价的机制。

**第三条** FTP 体系的核心作用：

- 把分支机构变成"零利率风险"的中间业务单位
- 由总行司库集中管理利率风险、流动性风险
- 引导业务结构（鼓励长期稳定存款、限制错配）
- 精确核算条线/产品/客户/分支机构的真实贡献度

## 第三章 业务条线 FTP 应用

### 3.1 存款业务（资金来源端）

**第七条** 分支机构吸收存款，按以下规则向总行司库出售资金：

| 存款类型 | FTP 出售价 |
|---|---|
| 活期存款 | FTP_活期（约 2.30%） |
| 通知存款 | FTP_7天 |
| 1 个月定期 | FTP_1月 |
| 3 个月定期 | FTP_3月 |
| 6 个月定期 | FTP_6月 |
| 1 年定期 | FTP_1年 |
| 2 年定期 | FTP_2年 |
| 3 年定期 | FTP_3年 |
| 5 年定期 | FTP_5年 |
| 大额存单 | 对应期限 + 5BP |
| 同业存款 | 对应期限 − 10BP |

**分支机构存款贡献 = 实际存款利率 vs FTP 出售价的正差**

### 3.2 贷款业务（资金运用端）

**第八条** 分支机构发放贷款，按以下规则向总行司库购入资金：

| 贷款品种 | FTP 购入价基准 |
|---|---|
| 短期流动资金贷款（≤ 1 年） | FTP_同期限 |
| 中期流贷（1~3 年） | FTP_3年 |
| 长期流贷（3~5 年） | FTP_5年 |
| 固定资产贷款（≤ 10 年） | FTP_对应期限 |
| 项目融资（≤ 15 年） | FTP_对应期限 + 流动性附加 5~15BP |
| 票据贴现（≤ 6 月） | FTP_6月 |
| 银行承兑（敞口部分） | FTP_对应期限 |
| 国内信用证（敞口部分） | FTP_对应期限 |
| 银行保函（敞口部分） | FTP_对应期限 + 5BP |

**分支机构贷款贡献 = 实际贷款利率 vs FTP 购入价的正差**

## 第四章 综合定价公式

**第九条** **客户综合定价公式**（核心，对外报价的内部成本测算）：

```
最低对外报价 = FTP 购入价
              + 资本占用成本（按风险加权资产 × 12%）
              + 风险溢价（按 PD/LGD 评级匹配）
              + 运营成本（约 30~50BP）
              + 税费（增值税 6%、附加税）
              + 目标利润率（≥ 1.5%）
```

**第十条** 不同评级客户的风险溢价（参考值）：

| 内部评级 | 风险溢价（BP） |
|---|---|
| AAA | 0~20 |
| AA | 20~50 |
| A | 50~100 |
| BBB | 100~180 |
| BB | 180~300 |
| B 及以下 | ≥ 300（或不予授信） |

**第十一条** **本行最低利率底线**（任何情况下不得突破）：

| 客户评级 | 最低对外利率 |
|---|---|
| AAA | LPR − 30BP |
| AA | LPR − 15BP |
| A | LPR + 0BP |
| BBB | LPR + 50BP |
| BB | LPR + 150BP |

## 第五章 期限错配与流动性附加

**第十二条** 业务期限与基准期限不一致时，按以下规则处理：

- 实际期限介于两个 FTP 期限之间 → 按线性插值
- 含权产品（提前还款权、回售权）→ 期权调整加点 5~30 BP
- 期限错配（资金来源期限 < 资产期限）→ 司库承担，分支机构无错配损益

**第十三条** **流动性附加**：

- 期限 > 5 年的资产业务，每延长 1 年加 3BP
- 期限 > 10 年的资产业务，每延长 1 年加 5BP
- 不可提前还款条款（lock-up）：根据期限给予 10~30BP 优惠

## 第六章 特殊业务定价

**第十四条** **普惠金融**：单户授信 ≤ 1000 万元的小微企业贷款，FTP 享受 30~50BP 减点（按监管两增两控政策）。

**第十五条** **绿色金融**：纳入本行绿色金融名单的项目，FTP 享受 20~40BP 减点。

**第十六条** **战略客户**：A 类战略客户，单笔金额 ≥ 1 亿元的授信，FTP 可享受 20BP 减点（须总行授信审批委员会批准）。

**第十七条** **跨境业务**：境内分行办理的跨境融资，FTP 应同时考虑外币汇率风险溢价。

## 第七章 FTP 调整与审批

**第十八条** FTP 减点权限：

| 减点幅度 | 审批层级 |
|---|---|
| ≤ 10BP | 一级分行行长 |
| 10BP < x ≤ 30BP | 一级分行行长办公会 |
| 30BP < x ≤ 50BP | 总行计划财务部 |
| > 50BP | 总行行长办公会 |

**第十九条** **严禁**以下行为：

- 通过 FTP 减点变相降低利率底线
- 以同业、关联方等名义套用低 FTP 价格
- 拆分业务规避审批权限
- 业务真实期限与系统记录期限不一致（变相套利）

---

## 第八章 绩效核算

**第二十条** **分支机构净利息收入** = 存款贡献 + 贷款贡献 + 中间业务收入 − 营业费用 − 风险拨备

**第二十一条** FTP 是经济资本回报率（RAROC）和经济增加值（EVA）核算的基础。

**第二十二条** 总行司库的经营成果体现为期限错配的"久期利差"和流动性管理收益。
