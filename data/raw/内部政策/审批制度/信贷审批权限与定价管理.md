# 信贷审批权限与贷款定价管理制度

---

## 第一章 审批权限体系

### 1.1 分级授权架构

建设银行实行"总行-一级分行-二级分行-支行"四级信贷管理体制：

```
总行信贷审批委员会
    │ 授权
    ├── 一级分行信贷审批委员会（省级）
    │       │ 授权
    │       ├── 二级分行信贷审批（地市级）
    │       │       │ 授权
    │       │       └── 支行行长审批
    │       └── 重大项目总行前置审批
    │
    └── 总行直接审批（特定客户/特定产品）
```

### 1.2 授信审批权限表

| 客户类型 | 授信金额 | 审批权限 |
|---------|---------|---------|
| 一般企业客户 | ≤500万 | 支行行长 |
| 一般企业客户 | 500万-2000万 | 二级分行信贷部 |
| 一般企业客户 | 2000万-5亿 | 一级分行审贷委 |
| 一般企业客户 | 5亿以上 | 总行授权/总行直批 |
| 小微客户（网络信贷） | ≤500万 | 系统自动审批 |
| 并购贷款 | 全部 | 总行审批 |
| 银团牵头行 | 全部 | 总行审批 |
| 集团客户统一授信 | 全部 | 总行统一管理 |

### 1.3 重大客户上报机制

以下情形须上报上级行审批：
- 授信主体为上市公司、央企、国资委重点监管企业
- 涉及跨地区、跨行业的大型并购重组
- 单户信贷余额超过一级分行核心资本10%
- 新开展的创新型业务或新类型担保方式

### 1.4 "双人操作"规定

以下信贷操作须两人以上共同完成：
- 贷款合同签订（经办人 + 主管或合规人员）
- 放款操作（经办 + 复核）
- 抵押物登记办理
- 贷后风险预警处置

## 第二章 贷款定价管理

### 2.1 定价基准

**人民币贷款**
- 短期（1年以内）：参考1年期LPR
- 中期（1-5年）：参考5年期LPR
- 长期（5年以上）：参考5年期LPR + 期限溢价

**外币贷款**
- SOFR（美元替代LIBOR后）+ 信用利差
- 参考各外币对应隔夜基准利率

### 2.2 内部资金转移定价（FTP）

内部定价基准（FTP率）由总行资金部每日发布，根据期限结构给出FTP曲线：

| 期限 | FTP率（示例，以LPR为基础） |
|------|------------------------|
| 1个月 | LPR - 20BP |
| 3个月 | LPR - 10BP |
| 6个月 | LPR |
| 1年 | LPR + 5BP |
| 3年 | LPR + 30BP |
| 5年 | LPR + 60BP |
| 7年 | LPR + 90BP |
| 10年 | LPR + 120BP |

### 2.3 客户贷款定价公式

```
贷款利率 = FTP基准 + 信用风险溢价 + 期限溢价 + 流动性溢价 + 目标利润

其中：
  信用风险溢价 = f(内部评级, 担保覆盖率, 行业系数, 客户贡献度)
  期限溢价     = 超出基准期限部分的加点
  流动性溢价   = 长期固定利率加点
  目标利润     = 行内FTP最低要求 + 业务中心营销溢价（可下浮）
```

**评级对应加点基准（BP）**

| 内部评级 | 信用加点范围 | 无担保加点 | 抵押加点 |
|---------|-----------|---------|---------|
| AAA | 0 - 20 | 0 - 30 | 0 - 10 |
| AA+ | 10 - 40 | 20 - 50 | 10 - 30 |
| AA | 20 - 60 | 40 - 80 | 20 - 50 |
| AA- | 40 - 80 | 70 - 120 | 40 - 70 |
| A+ | 60 - 110 | 100 - 150 | 60 - 100 |
| A | 80 - 140 | 120 - 180 | 80 - 120 |
| BBB | 130 - 200 | 180 - 260 | 130 - 180 |
| BB | ≥250 | ≥320 | ≥250 |
| B及以下 | 原则上不予授信 | - | - |

### 2.4 定价底线管理

**硬底线（不得突破）**
- 人民币贷款利率不得低于同期LPR - 30BP（监管底线）
- 对公贷款综合定价（含中间业务）须覆盖FTP成本
- 小微企业贷款不得设置附加费用

**优惠定价条件**
以下客户可申请优惠定价：
- 绿色信贷客户：可享受10-20BP优惠
- 科技型企业（专精特新）：可享受20-30BP优惠
- 存量优质客户（内部评级AA+以上，合作满3年）：定价下浮权限放开
- 银团贷款（市场化竞争）：参考市场银团利率水平

### 2.5 综合定价与交叉销售

**客户综合收益测算**

```
综合收益 = 贷款利息收入 + 手续费收入 + 结算收入 + 理财收入 + ...

综合利率 = 综合收益 / 客户总授信余额

最低综合利率要求：
  - 一般客户：LPR + 50BP
  - 优质客户（AA+）：LPR + 20BP
  - 重要客户（AAA，集团授信）：LPR + 0BP
```

**交叉销售捆绑要求（禁止强制）**
- 可通过优惠定价引导客户使用配套结算/投行/理财服务
- **严禁**：以贷款为条件强制搭售任何产品
- **严禁**：以任何形式收取"顾问费"、"咨询费"变相收费

## 第三章 集团授信管理

### 3.1 集团客户统一授信

对于同一集团下多家成员企业，实行"集团统一授信+成员企业分配使用"机制：

**统一授信上限**
```
集团总授信上限 = min(
    集团合并净资产 × 2倍,
    集团合并主营收入 × 40%,
    银行对同一集团风险暴露上限（核心资本15%）
)
```

**子公司额度管理**
- 各子公司使用额度不得超过总集团额度
- 子公司担保互保链须完整评估风险暴露
- 关联方交易须穿透核查，防止套取银行资金

### 3.2 大额风险暴露管理（巴塞尔III要求）

对同一交易对手的大额风险暴露不得超过核心一级资本的**15%**（关联客户合并计算25%）。

超出上限须在规定时间内压降，或通过证券化、信用风险缓释等方式降低敞口。
