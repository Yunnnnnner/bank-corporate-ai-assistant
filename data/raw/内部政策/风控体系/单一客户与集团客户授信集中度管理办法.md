# 单一客户与集团客户授信集中度管理办法

**发布部门**：本行风险管理部
**发布日期**：2025-09-10
**生效日期**：2025-10-01
**版本**：v2.1

---

## 第一章 总则

**第一条** 为防范本行授信集中度风险，依据《商业银行法》第三十九条、《商业银行大额风险暴露管理办法》（银保监会令2018年第1号）及监管要求制定本办法。

**第二条** 本办法所称授信集中度，是指单一客户、集团客户、行业、区域、产品在本行授信总额中所占比例。

**第三条** 集中度管理遵循"分散风险、限额硬约束、动态监测、违规问责"原则。

---

## 第二章 监管硬约束指标（核心红线）

**第四条** 监管约束的硬指标如下，**任何情况下不得突破**：

| 集中度类型 | 监管上限 | 本行内控线 |
|---|---|---|
| 单一非同业客户授信 / 一级资本净额 | ≤ 15% | 12% |
| 单一同业客户授信 / 一级资本净额 | ≤ 25% | 20% |
| 集团客户授信 / 一级资本净额 | ≤ 15% | 13% |
| 关联客户授信 / 一级资本净额 | ≤ 15%（综合）/ 5%（单一关联方） | 4% |
| 全部并购贷款余额 / 一级资本净额 | ≤ 50% | 40% |
| 单一借款人并购贷款余额 / 一级资本净额 | ≤ 5% | 4% |
| 房地产业贷款余额 / 各项贷款余额 | ≤ 22.5%（中型银行 15%） | 14% |
| 个人住房贷款余额 / 各项贷款余额 | ≤ 17.5%（中型银行 12%） | 11% |
| 政府融资平台余额 / 一级资本净额 | 名单制管理 | 名单制 |

**注**：实际监管上限以最新银保监会规章为准，本行依据监管口径调整。

---

## 第三章 行业集中度

**第五条** 行业限额：本行根据国民经济行业分类，每年由总行授信审批委员会核定 23 个一级行业的限额上限。

| 行业 | 集中度上限（占各项贷款比例） |
|---|---|
| 制造业（综合） | ≤ 25% |
| ├ 钢铁、煤炭、水泥（产能过剩） | ≤ 1.5%（名单制） |
| ├ 新能源、新材料、高端制造 | 鼓励，限额上浮 30% |
| 房地产开发 | ≤ 14% |
| 建筑业 | ≤ 8% |
| 批发和零售业 | ≤ 12% |
| 交通运输、仓储和邮政业 | ≤ 8% |
| 信息传输、软件和信息技术服务业 | ≤ 5% |
| 金融业（同业） | ≤ 25% |
| 租赁和商务服务业 | ≤ 8% |
| 农、林、牧、渔业 | ≤ 4% |
| 采矿业 | ≤ 3%（含名单制约束） |
| 电力、热力、燃气及水生产 | ≤ 6% |
| 教育、卫生、文化（普惠） | 鼓励，限额单列 |
| 其他行业（综合） | ≤ 30% |

**第六条** **限制类行业名单**：钢铁、电解铝、平板玻璃、煤炭、水泥、船舶、电石、烧碱、多晶硅、化纤等"两高一剩"行业实行名单制管理，仅对名单内龙头企业授信。

---

## 第四章 区域集中度

**第七条** 一级分行授信余额占本行各项贷款余额的比例：

- 单一一级分行 ≤ 15%
- 同一省份所有分行合计 ≤ 25%
- 经济发达地区（粤港澳大湾区、长三角、京津冀）合计 ≥ 50%（导向性目标）

**第八条** 跨境授信集中度：

- 单一国家/地区授信 / 一级资本净额 ≤ 5%
- 高风险国别（按本行国别风险评级 5 档及以上）授信 / 一级资本净额 ≤ 1%

---

## 第五章 产品集中度

**第九条** 产品授信限额：

| 产品 | 限额上限（占总授信比例） |
|---|---|
| 流动资金贷款 | ≤ 50% |
| 固定资产贷款 | ≤ 30% |
| 项目融资 | ≤ 15% |
| 并购贷款 | ≤ 8% |
| 银行承兑汇票 | ≤ 25% |
| 国内信用证 | ≤ 8% |
| 银行保函 | ≤ 10% |
| 贸易融资 | ≤ 8% |
| 供应链金融 | ≤ 12% |
| 票据贴现 | ≤ 15% |

---

## 第六章 集中度监测与报告

**第十条** 监测频率：

- 单一客户、集团客户、关联方：T+1 实时监测
- 行业、区域：月度监测
- 产品：季度监测

**第十一条** 触限处置：

- 达到限额 80% → 黄色预警，限制新增同类授信
- 达到限额 90% → 橙色预警，停止新增并启动压降计划
- 达到限额 100% → 红色预警，立即停止新增，3 个月内必须压降至 90% 以下

**第十二条** 报告制度：

- 月度集中度报表 → 总行风险管理委员会
- 季度集中度分析报告 → 董事会风险管理委员会
- 年度集中度评估报告 → 董事会、监管机构

---

## 第七章 突破限额的应急处置

**第十三条** 因不可抗力或外部事件导致超限的，应在 5 个工作日内向风险管理委员会报告，制定 3 个月内的压降计划。

**第十四条** **严禁**通过以下方式规避集中度管理：

- 拆分授信主体（化整为零）
- 通过表外业务（保函、承兑、信用证、理财）替代表内授信
- 通过通道业务（信托、券商资管）转嫁风险
- 关联方分散授信
- 跨境联动放大授信

---

## 第八章 附则

**第十五条** 本办法由本行风险管理部负责解释。

**第十六条** 本办法自 2025 年 10 月 1 日起施行。
