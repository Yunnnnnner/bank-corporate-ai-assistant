# 投行债券承销业务操作手册

**发布部门**：本行投资银行部
**发布日期**：2025-12-15
**生效日期**：2026-01-01
**版本**：v3.1

---

## 第一章 总则

**第一条** 为规范本行作为承销商开展企业债券承销业务，依据《证券法》《公司法》《非金融企业债务融资工具管理办法》（人行令〔2008〕第1号）、银行间市场交易商协会相关自律规则、《商业银行表外业务风险管理办法》及本行《投行业务管理办法》制定本手册。

**第二条** 本手册所称债券承销业务，是指本行作为承销商或承销团成员，向投资者销售企业（公司）发行的债券，并向发行人提供方案设计、发行定价、簿记建档、登记结算等服务的业务。

---

## 第二章 业务范围

**第三条** 本行承销的主要债券品种：

### 银行间市场（交易商协会注册）
1. **超短期融资券（SCP）**：期限 ≤ 270 天
2. **短期融资券（CP）**：期限 ≤ 1 年
3. **中期票据（MTN）**：期限通常 3~10 年
4. **永续票据（含权 MTN）**：无固定到期日
5. **资产支持票据（ABN）**：以基础资产支持发行
6. **定向工具（PPN）**：私募定向发行
7. **熊猫债**：境外机构在境内发行的人民币债券

### 交易所市场（证监会管辖）
8. **公司债**：上交所/深交所发行
9. **企业债**：发改委审核
10. **资产支持证券（ABS/CMBS/RMBS）**

### 国际债券
11. **海外债**：境内主体在境外发行的外币债券
12. **绿色债券、可持续债券**：符合 GBP/SLB 原则

## 第三章 业务准入

**第四条** **发行人准入条件**：

（一）依法设立，连续经营 3 年以上（地方国企、政府平台、央企子公司可放宽）；
（二）最近 3 个会计年度连续盈利（永续/可转债等品种除外）；
（三）信用评级 AA- 及以上（不同品种要求不同）；
（四）资产负债率合理（产业类 ≤ 70%，金融类 ≤ 90%）；
（五）拟发行规模不超过净资产 40%（公司债）；
（六）不存在重大违法违规、债务违约记录。

**第五条** **本行不予承销的情形**：

- 国家产业政策淘汰类企业
- 募集资金用于股票投机、土地储备、商品房开发未取得四证、违规建设项目
- 发行人涉及重大诉讼、违法被立案调查
- 募集资金用于偿还非本行金融机构有息债务（专项审批）
- 关联方循环发债套取资金

---

## 第四章 承销方式

**第六条** 承销方式分类：

1. **包销**：本行包销全部债券，未售完部分由本行自有资金承担（**风险极高，须升级审批**）
2. **余额包销**：本行包销发行规模与销售额之差额
3. **代销**：仅作分销，未售完部分由发行人收回
4. **联合承销**：与其他承销商组建承销团，按比例分销

**第七条** 本行优先采用**联合主承销 + 代销/余额包销**模式，单独包销须由总行投行委员会审批。

## 第五章 业务流程

**第八条** 完整流程（以银行间 MTN 为例）：

```
① 立项 → 内部立项委员会审议
② 尽职调查（财务、法律、行业、合规）
③ 制作申报材料（注册文件、募集说明书、评级报告、法律意见书）
④ 报送交易商协会注册
⑤ 注册接受 → 取得注册通知书
⑥ 簿记建档（询价、定价、配售）
⑦ 缴款发行
⑧ 登记结算（中央国债登记结算公司 / 上海清算所）
⑨ 上市流通
⑩ 存续期管理（受托管理、信息披露、回售/兑付）
```

**第九条** 关键时限：

- 立项至注册接受：通常 60 个工作日
- 簿记建档至上市：1~5 个工作日
- 信息披露：注册后每年定期披露 + 重大事项临时披露

## 第六章 簿记建档与定价

**第十条** **簿记建档**是确定债券发行价格、利率的核心环节：

- 公开询价区间（参考同评级、同期限债券二级市场水平 + 风险溢价）
- 投资者提交申购订单
- 按"价格优先、时间优先"原则配售
- 主承销商（本行）有权调整最终票面利率

**第十一条** 定价考量因素：

- 同评级、同期限国债收益率（基准）
- 信用风险溢价
- 流动性溢价
- 期权价值（含权债券）
- 当前市场利率走势
- 发行规模与市场容量

**第十二条** 价格区间监控：

- 簿记利率不得高于上限（避免发行人融资成本过高）
- 簿记利率不得低于下限（避免市场扭曲）
- 突破区间须报总行投行委员会备案

## 第七章 投资者管理

**第十三条** 投资者类型：

- 银行间市场：商业银行、保险机构、券商自营、基金、信用社、财务公司、QFII/RQFII
- 交易所市场：合格投资者（机构 + 高净值个人）

**第十四条** **关联方申购管理**：

- 本行自营资金原则上不参与本行主承销债券的簿记申购
- 本行理财、资管产品申购须满足"独立决策、市场化定价、不构成承销商隐性兜底"
- 关联方申购比例 > 30% 须重点关注

---

## 第八章 募集资金用途监督

**第十五条** 募集资金使用应严格遵守募集说明书披露的用途，不得变相用于：

- 股票投机
- 衍生品投机
- 商品房开发（未取得四证）
- 偿还本行贷款（构成关联交易）
- 借给关联方违规使用

**第十六条** 用途监督机制：

- 设立募集资金专项账户
- 资金使用须由本行联签
- 季度信息披露 + 受托管理人持续核查
- 用途变更须经持有人会议同意

---

## 第九章 风险管理

**第十七条** 主要风险点：

| 风险类型 | 描述 | 防控措施 |
|---|---|---|
| 市场风险 | 利率/汇率波动导致价格变化 | 簿记建档锁定价格 |
| 包销风险 | 未售完部分本行承担 | 限制包销规模 ≤ 单期 30% |
| 信用风险 | 发行人违约 | 发行人准入、评级跟踪 |
| 操作风险 | 流程瑕疵、信披错误 | 双线核对、合规复核 |
| 法律风险 | 募集说明书重大遗漏 | 法律意见书 + 交叉复核 |
| 声誉风险 | 大型违约事件 | 选择高评级发行人 |

**第十八条** 限额管理：

- 单一发行人承销额 ≤ 该发行人净资产 40%
- 单期承销额 ≤ 本行一级资本 5%
- 本行包销/余额包销总余额 ≤ 一级资本 10%
- 本行承销同业债券集中度严格控制

---

## 第十章 收费标准

**第十九条** 承销费率（按发行规模 × 期限因子）：

| 品种 | 费率参考区间 |
|---|---|
| 超短期融资券（SCP） | 0.05%~0.15% |
| 短期融资券（CP） | 0.10%~0.30% |
| 中期票据（MTN） | 0.30%~0.80% |
| 永续票据 | 0.50%~1.00% |
| 公司债 | 0.30%~0.70% |
| 资产支持票据 | 0.50%~1.50% |
| 海外债 | 0.30%~1.00% |

**第二十条** 实际费率由市场议价决定，必须覆盖本行尽调成本、合规成本及合理利润。
