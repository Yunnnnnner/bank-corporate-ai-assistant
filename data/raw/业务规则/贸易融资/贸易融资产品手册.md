## 第一章 信用证业务

### 1.1 信用证基本逻辑

**定义**：银行依照客户（申请人）的要求，向受益人开立的，保证在符合信用证条款的单据提交时付款的书面承诺。

**三大法律原则**
1. **独立性原则**：信用证独立于基础贸易合同，银行只审核单据，不管货物
2. **单据相符原则**：只要单据与信用证要求"严格相符"，银行必须付款
3. **欺诈例外原则**：有充分欺诈证据时，可申请法院止付

### 1.2 信用证种类

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 即期信用证 | 见单立即付款 | 卖方话语权强 |
| 远期信用证 | 承兑后到期付款（30-180天） | 买方需要账期 |
| 可转让信用证 | 受益人可转让给第三方 | 中间商贸易 |
| 背对背信用证 | 以原证为基础开立新证 | 中间商不转让情况 |
| 保兑信用证 | 第三方银行加保兑承诺 | 开证行信用不足时 |
| 备用信用证（SBLC） | 类似保函，作为还款备用 | 担保用途为主 |

### 1.3 开证业务操作规范

**开证申请条件**
- 进口商在建行开立结算账户
- 纳入银行综合授信额度（开证保证金+授信覆盖）
- 提供贸易合同、商业发票（PI）
- 对应业务许可证（进口许可证、外汇使用批准等）

**开证审核要点**
1. 开证申请书与贸易合同一致性
2. 证面金额与实际贸易背景匹配
3. 装运条款合理性（装运期、效期）
4. 单据要求合理性（单据名称、份数、内容要求）
5. 特殊条款识别（软条款识别——防止开证行利用软条款拒付）

**软条款识别清单（须警示）**
- "货物经检验合格后方可提示单据"
- "需开证申请人出具检验合格证"
- "装船通知需经申请人书面确认"
- "大副收据需申请人签字"

### 1.4 议付/审单操作

**标准审单时间**：不超过5个工作日（UCP600规定）

**常见单据不符点**
| 不符点类型 | 具体示例 |
|---------|---------|
| 提单问题 | 收货人不符、背书有误、港口不符 |
| 发票问题 | 金额超证、品名与信用证不符 |
| 产地证问题 | 签发机构不符、内容与发票不一致 |
| 期限问题 | 晚装船、单据提示逾期 |
| 格式问题 | 单据未签字、份数不足 |

## 第二章 进出口押汇

### 2.1 进口押汇

**定义**：进口商收到货物但暂时无力全额付款时，以进口单据（提单）为质押，由银行垫款付给国外银行，进口商随后向银行还款。

**业务逻辑**
```
进口商申请 → 银行垫款付汇 → 进口商取得提单提货 → 货物销售 → 还款给银行
```

**关键风控**
- 质押物（提单）的控制（银行持有正本提单）
- 进口货物的价格波动风险（商品价格暴跌）
- 进口商销售能力评估

### 2.2 出口押汇

**定义**：出口商向银行提交符合信用证要求的单据，银行按单据金额的一定比例（通常90%-95%）预付货款。

**优势**：出口商无需等待国外银行付款（通常15-30天），可立即获得资金

**风险**：银行承担"单据不符被拒付"的风险（有追索权押汇需出口商承担）

## 第三章 保函业务

### 3.1 保函类型与适用场景

| 保函类型 | 定义 | 适用场景 |
|---------|------|---------|
| 投标保函 | 保证投标人不撤标或中标后签约 | 招投标项目 |
| 履约保函 | 保证合同履行 | 工程建设、货物供应 |
| 预付款保函 | 保证买方已付预付款在合同不履行时退还 | 大额合同预付款 |
| 付款保函 | 保证申请人支付货款 | 买方信用不足时 |
| 融资保函 | 担保借款人还款 | 银行借款担保 |
| 关税保函 | 海关保证 | 暂时进口货物 |

### 3.2 独立保函（见索即付）

**特点**：受益人一旦提出索赔并符合保函规定条件，银行必须立即付款，不管申请人与受益人之间有无争议。

**独立性原则**：保函独立于基础合同，与保证不同。

**欺诈例外**：有充分证据证明受益人欺诈性索赔时，申请人可向法院申请禁令。

### 3.3 保函申请条件

- 申请人在建行有效授信（保函金额纳入授信敞口）
- 基础合同真实、合法
- 保函条款合理（金额、期限、索赔条件）
- 提供反担保（抵押/质押/保证金）

## 第四章 福费廷（Forfaiting）

### 4.1 定义

福费廷是出口商将其持有的、经进口商承兑或银行担保的远期票据（通常为信用证项下汇票）出售给银行（福费廷商），从而提前收款的融资方式。

**关键特征**：**无追索权**（Non-recourse）——银行承买后，不得再向出口商追索

### 4.2 业务流程

```
出口商 → 发货 → 提示单据 → 取得承兑汇票（开证行/确认行承兑）
    │
    ├── 将汇票无追索出售给建行（福费廷）
    │   建行支付：票面金额 - 贴现利息 - 手续费
    │
建行持有汇票 → 到期向承兑行索款（全额）
```

### 4.3 适用条件

- 远期信用证项下汇票（期限30天-7年）
- 必须有承兑行背书（信用替代）
- 承兑行须是建行认可的银行（通常为国际知名银行）

### 4.4 定价

```
福费廷利率 = LIBOR/SOFR/SHIBOR + 承兑行国家风险溢价 + 信用风险溢价 + 利润
实际融资金额 = 票面金额 / (1 + 利率 × 期限/360)
```

## 第五章 跨境人民币业务

### 5.1 跨境人民币结算

**适用范围**
- 跨境贸易人民币结算（商品+服务）
- 直接投资（FDI/ODI）项下人民币结算
- 跨境融资人民币使用

**优势**
- 消除汇率风险
- 降低换汇成本
- 提升人民币国际化

### 5.2 自由贸易账户（FT账户）

自贸区内企业可开立FT账户，享受跨境资金自由流动便利：
- FT账户资金可自由跨境转移（不受外汇管制）
- FT账户与境内账户之间的转移需要业务背景支撑
- 用于实现离岸资金归集和境内外联动融资
