# 资产证券化（ABS）操作手册
## 信贷资产证券化（CLO/RMBS）+ 企业ABS + 不良ABS

---

## 第一章 资产证券化概述

### 1.1 产品分类体系

```
资产证券化
├── 信贷ABS（银行间市场，人民银行监管）
│   ├── CLO：企业贷款资产支持证券
│   ├── RMBS：个人住房抵押贷款支持证券
│   ├── CMBS：商业房产抵押贷款支持证券
│   ├── Auto-ABS：汽车贷款证券化
│   ├── 消费贷ABS：个人消费贷款
│   └── NPL-ABS：不良贷款证券化
│
├── 企业ABS（交易所市场，证监会监管）
│   ├── 供应链ABS（应收账款）
│   ├── 融资租赁ABS
│   ├── 基础设施收费权ABS
│   └── REITs（基础设施公募）
│
└── ABN（资产支持票据，银行间市场）
    ├── 项目收益ABN
    └── 供应链ABN
```

### 1.2 核心机制：破产隔离

```
发起行（原始权益人）
    │ 真实销售（True Sale）
    │ 资产从发起行资产负债表剥离
    ▼
信托计划/专项计划（SPV）
    │ 以基础资产现金流为支撑
    │ 发行分层证券
    ▼
投资者购买证券
    │ 优先级（AAA）获优先受偿
    │ 次级由发起行持有≥5%
```

**破产隔离效果**：即使发起行破产，SPV中的资产不纳入清算，投资者权益不受影响。

## 第二章 CLO（企业贷款证券化）操作全流程

### 2.1 基础资产筛选标准

**入池资格（Eligibility Criteria）**

| 筛选维度 | 准入条件 |
|---------|---------|
| 贷款状态 | 正常类（五级分类），非关注类以下 |
| 逾期状态 | 无逾期记录，或逾期≤30天 |
| 借款人类型 | 中国境内企业法人 |
| 担保方式 | 有抵押/质押优先；信用类占比≤30% |
| 剩余期限 | 3个月-10年 |
| 贷款余额 | 单笔不超过资产池总额的5% |
| 行业限制 | 房地产、地方政府平台占比≤30% |
| 利率类型 | 固定利率和浮动利率均可，但须在报告中披露 |

**剔除标准（Exclusion Criteria）**
- 借款人内部评级低于BBB的贷款
- 已进入司法程序的贷款
- 关联方贷款（与发起行关联）
- 已作其他质押/托管安排的贷款

### 2.2 交易结构设计

**分层方案示例**

```
┌─────────────────────────────────────┐
│          优先A级（AAA）     70%      │  ← 最优先受偿，评级最高，利率最低
├─────────────────────────────────────┤
│          优先B级（AA+）     12%      │  ← 次优先
├─────────────────────────────────────┤
│           次优先级（A）      8%      │
├─────────────────────────────────────┤
│         次级（未评级）       10%     │  ← 发起行持有（≥5%监管要求）
│         超额利差账户                  │  ← 缓冲垫
└─────────────────────────────────────┘
```

**信用增级方案**
1. **超额抵押（Over-collateralization）**：资产池总面值 > 证券发行总额，差额为安全垫
2. **超额利差（Excess Spread）**：资产池利息收入 > 证券付息成本，差额用于覆盖损失
3. **现金储备账户（Reserve Account）**：初始注入3%-5%的资产池余额
4. **流动性支持**：银行提供流动性备用信贷额度，覆盖临时性资金缺口

### 2.3 发行全流程时间线

**阶段一：准备期（约3-4个月）**
```
Week 1-4:   确定证券化目标与规模，聘请中介团队
Week 5-8:   基础资产尽职调查（法律+会计+评级）
Week 9-12:  交易结构设计与优化，信托合同起草
Week 13-16: 信用评级机构评级（初稿→问答→终稿）
```

**阶段二：审批与发行期（约1-2个月）**
```
Week 17-18: 向人民银行信贷资产证券化试点管理办公室备案
Week 19-20: 路演推介（机构投资者：银行、保险、基金）
Week 21:    簿记建档、定价（利差确定）
Week 22:    证券交割、资金划付
Week 22+1:  证券在银行间债券市场挂牌交易
```

**阶段三：存续期管理（贷款剩余期限）**
- 月度：服务机构报告（还款/逾期/违约数据）
- 季度：评级机构监控报告
- 事件驱动：违约事件、早偿事件及时披露

### 2.4 各中介机构职责分工

| 机构 | 职责 | 费用基准 |
|------|------|---------|
| 主承销商（建行投行部/承销商） | 组织发行、路演、簿记 | 承销费 0.1%-0.3% |
| 受托机构（信托公司） | 设立信托、管理SPV | 管理费年费 |
| 律师事务所 | 法律意见书、合同审核 | 固定服务费 |
| 会计师事务所 | 资产池审计、财务报告 | 审计费 |
| 信用评级机构（中诚信/联合） | 初评+跟踪评级 | 评级费+年费 |
| 资产服务机构（通常为发起行） | 贷款日常服务 | 服务费0.1%-0.3% |
| 托管行 | 资金托管 | 托管费 |

### 2.5 监管资本节约测算

**原始权益人的核心动机**

```
若保留在表内（CLO资产）：
  风险权重 = 100%（企业贷款）
  资本占用 = 贷款余额 × 100% × 8%（最低资本要求）

证券化后（仅保留次级5%）：
  资本占用 = 次级规模 × 1250% × 8%
  实际节约资本 = 95% × 贷款余额 × 8%

示例：100亿CLO发行
  传统方式资本占用：100亿 × 8% = 8亿
  证券化后资本占用：5亿（次级） × 1250% × 8% = 5亿
  节约资本：3亿（37.5%节约）
```

## 第三章 不良ABS（NPL-ABS）

### 3.1 业务背景与政策依据

2016年银监会重启不良贷款证券化试点，允许国有大行和股份制银行以不良贷款为基础资产发行ABS，主要目的是加快不良资产出表、补充流动性。

### 3.2 基础资产特殊处理

**不良资产池评估**
- 不良贷款历史回收率分析（过去3-5年）
- 抵押物现值评估（专业评估机构）
- 现金流预测（蒙特卡洛模拟）

**分层定价逻辑**
由于不良贷款本金损失确定，优先级规模通常仅为资产池面值的**30%-50%**（远低于正常CLO的70%+），剩余由次级吸收。

### 3.3 不良ABS特殊风险

- **服务机构风险**：催收效率直接影响现金流，须对服务机构能力审核
- **法律风险**：批量转让合规性（须满足相关规定）
- **道德风险**：发起行可能将质量最差的不良打包出售
- **处置周期风险**：诉讼/拍卖程序漫长，影响现金流时序

## 第四章 供应链ABS（交易所市场）

### 4.1 交易结构

```
核心企业（付款人）
    │── 确认对供应商的应付账款
    │
专项计划（券商管理人）
    │── 受让应收账款债权
    │── 发行ABS
    │
投资者
```

### 4.2 核心差异

相比CLO，供应链ABS基础资产为**应收账款**（而非贷款），信用取决于**核心企业主体信用**，而非分散的借款人群体。

**优势**：结构简单、审批快（交易所约20个工作日）、成本低
**劣势**：高度依赖单一核心企业信用、集中度风险高

### 4.3 准入标准

**核心企业要求**
- 主体评级AA及以上（公开评级）
- 非房地产类（监管趋严）
- 有稳定的上下游贸易关系
- 同意参与确权流程

**基础资产（应收账款）要求**
- 账期不超过1年
- 非关联方交易
- 不存在权利受限（无抵押、无查封）
- 增值税发票已开具
