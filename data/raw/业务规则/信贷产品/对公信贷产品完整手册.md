# 对公信贷产品完整操作手册
## 参照建设银行公司信贷业务规范

---

## 第一章 流动资金贷款

### 1.1 产品定义与适用范围

**产品定义**
流动资金贷款是银行向企（事）业法人或国家规定可以作为借款人的其他组织，发放的用于借款人日常生产经营周转的本外币贷款。

**适用范围**
- 采购原材料、零部件、燃料动力
- 支付工资、水电费等经营性支出
- 支付税款、社保等法定支出
- 偿还正常经营产生的短期债务

**不适用范围**（严禁用途）
- 固定资产投资、股权投资
- 房地产开发
- 归还其他银行贷款（除批准的置换业务外）
- 股票、期货等金融市场投机

### 1.2 申请条件

**基本条件（全部满足）**
1. 依法注册，营业执照在有效期内
2. 具有良好的信用记录，无重大违约历史
3. 具有合理的流动资金借款需求
4. 有稳定的还款来源
5. 在本行开立结算账户（原则上）
6. 无不良贷款（含关注类须评估）

**财务条件（参考标准）**
| 指标 | 一般要求 | 优质客户放宽 |
|------|---------|------------|
| 资产负债率 | ≤70% | ≤75% |
| 流动比率 | ≥1.0 | ≥0.8 |
| 利息保障倍数 | ≥1.5倍 | ≥1.2倍 |
| 主营业务收入（近两年） | 无持续下降 | 下降≤20% |

### 1.3 期限与额度规则

**期限规定（国家金融监管总局2024年修订版）**
- 流动资金贷款期限原则上不超过**3年**
- 一年以内为短期流动资金贷款
- 超过一年须论证长期流动资金需求合理性

**额度测算公式**
```
流动资金贷款需求额度 = 营运资金缺口
= 流动资产 - 非银行流动负债
≈ (年销售收入 / 周转次数) × (1 - 自有资金比例)

周转次数 = 360 / (存货周转天数 + 应收账款周转天数 - 应付账款周转天数)
```

**实操示例**
某制造企业：
- 年销售收入：1亿元
- 存货周转天数：60天，应收账款周转天数：45天，应付账款周转天数：30天
- 周转天数合计：60+45-30 = 75天
- 需求额度 ≈ 1亿 × (75/360) × 80% ≈ **1,667万元**

### 1.4 全流程操作规范

**阶段一：贷前调查（不少于5个工作日）**

现场调查要素：
1. 经营场所核查（是否与营业执照地址一致）
2. 库存、设备核查（与财务报表比对）
3. 主要交易对手访谈（3家以上）
4. 账户流水核查（近12个月）
5. 税务申报核查（与收入交叉验证）

调查报告必备内容：
- 客户基本信息及经营状况
- 贷款需求合理性分析
- 还款来源分析（第一还款来源+第二还款来源）
- 担保分析
- 风险提示与缓释措施
- 综合收益测算

**阶段二：客户评级与授信申请**

客户内部评级维度：
| 维度 | 权重 | 评分项 |
|------|------|-------|
| 财务状况 | 40% | 盈利能力、偿债能力、营运能力、成长能力 |
| 管理水平 | 20% | 股权结构、管理团队、内控制度 |
| 行业地位 | 20% | 市场份额、竞争壁垒、行业周期 |
| 信用记录 | 20% | 历史还款、征信报告、诉讼记录 |

评级结果：AAA/AA/A/BBB/BB/B/CCC/CC/C/D（十级）

**阶段三：审批（按权限分级）**

| 授信金额 | 审批权限 |
|---------|---------|
| 500万以下 | 支行行长审批 |
| 500-3000万 | 分行信贷部审批 |
| 3000万-5亿 | 分行审贷委审批 |
| 5亿以上 | 总行审批/总行授权 |
| 并购贷款/银团 | 总行审批 |

**阶段四：合同签订与放款**

受托支付要求（2024年新规）：
- 单笔≥300万元的流动资金贷款，须采用受托支付
- 借款人直接支付条件：借款人提供充分理由，且单笔支付≤100万
- 资金必须进入贷款合同约定的账户

**阶段五：贷后管理**

检查频率：
- 正常类贷款：每季度一次
- 关注类贷款：每月一次
- 不良贷款：每周跟踪

预警信号清单：
□ 账户资金流入同比下降30%以上
□ 征信查询次数当月超过3次
□ 核心高管离职或失联
□ 主要供应商/客户出现纠纷
□ 法院冻结令或查封通知
□ 纳税额度连续2月下降50%以上
□ 抵押物评估价值下降超过30%

## 第二章 固定资产贷款

### 2.1 产品定义

向企业法人发放的，用于固定资产投资（含购置、建造、改扩建）的中长期贷款。

### 2.2 关键监管约束（固定资产贷款管理办法2024）

- 贷款比例：不超过项目总投资的**80%**（自有资金≥20%）
- 期限：与项目建设周期和现金流匹配，原则上不超过**10年**
- 还款计划：须制定与项目现金流匹配的还款计划
- 资金监管：全部采用受托支付，银行监控资金流向

### 2.3 项目评估核心要素

**技术可行性**
- 工艺技术是否成熟可靠
- 设备来源是否有保障
- 建设方案是否合理

**财务可行性**
- 项目财务内部收益率（FIRR）≥行业基准收益率
- 贷款偿还期（包括建设期）≤贷款期限
- 项目资本金收益率（ROE）≥12%

**市场可行性**
- 产品需求预测（乐观/基准/悲观三种情景）
- 行业竞争格局分析
- 客户订单支撑程度

## 第三章 并购贷款

### 3.1 监管要求（银监发〔2008〕84号及后续修订）

**强制性约束**
1. 并购贷款占并购价款比例：**不超过60%**
2. 并购方自有资金比例：**不低于40%**
3. 贷款期限：**不超过7年**
4. 贷款用途：仅限支付并购交易价款
5. 必须进行并购目标的独立尽职调查

**负面清单（不得发放并购贷款）**
- 并购方净资产低于贷款金额
- 目标企业资产负债率超过70%
- 贷款对象为非上市公司且无审计报告
- 借款人无并购整合能力
- 并购资金可能被转移或挪用

### 3.2 尽职调查重点

**法律层面**
- 并购协议/股权转让协议合法性
- 目标公司股权结构清晰（无隐名股东/代持）
- 监管审批（反垄断/国资委/行业主管部门）

**财务层面**
- 目标企业近3年审计财务报告
- 或有负债（未决诉讼、担保责任）
- 税务风险（税收优惠依赖性、历史欠税）

**经营层面**
- 核心团队稳定性与留存方案
- 核心技术/客户/供应商迁移风险
- 协同效应量化分析

## 第四章 银团贷款

### 4.1 定义

两家或两家以上银行依据同一贷款协议，按照商定比例和条件向同一借款人提供贷款的行为。

### 4.2 参与方架构

| 角色 | 主要职责 | 费用 |
|------|---------|-----|
| 牵头行（Mandated Lead Arranger, MLA） | 组织银团、谈判条款、起草IM | 安排费 0.3-1% |
| 代理行（Agent Bank） | 贷款发放/回收、贷后管理 | 代理费年费 |
| 参团行 | 按协议比例放款 | 利差收入 |
| 担保代理行 | 管理抵押/质押品 | 担保代理费 |

### 4.3 适用场景

- 单一银行超出大额风险暴露限额（≤核心一级资本15%）
- 金额大、期限长的基础设施/地产项目
- 跨境银团（中资+外资行参与）
- 政策性项目（"一带一路"）

### 4.4 银团贷款协议核心条款

**财务约束条款（Financial Covenants）**
- 资产负债率不超过XX%（测试频率：季度/年度）
- 利息保障倍数不低于XX倍
- 最低流动性比率要求

**违约事件（Events of Default）**
- 付款违约（Grace Period：5-15个工作日）
- 陈述与保证失实
- 财务约束条款违反
- 交叉违约（借款人其他债务）
- 控制权变更

## 第五章 网络信贷（数字信贷）

### 5.1 结算云贷

**产品特点**
基于企业在建行的结算账户数据（资金流量、流水规律）建立评分模型，自动审批授信。

**准入条件**
- 在建行开立结算账户满1年
- 近1年内月均结算金额有效

**额度计算模型**
```
信用额度 = min(
    月均结算金额 × 系数,   # 结算流量模型
    200万元                # 产品上限
)
系数根据账龄、稳定性、增长率动态调整（0.3-1.0）
```

**全程线上操作**：申请→审批→放款→还款，T+0实时响应

### 5.2 云税贷

**准入条件**
- 纳税信用等级：A级（最高额度500万）、B级（300万）、M级（200万）
- 近24个月内无欠税/滞纳金记录
- 近3年无涉税违法行为

**额度计算模型**
```
贷款额度 = min(
    近12个月完税额 × 10,  # 主要模型
    纳税信用等级对应上限
)
```

### 5.3 云电贷

**准入条件**
- 在国家电网/南方电网缴纳电费满2年
- 用电规模达标（按地区不同标准）
- 无拖欠电费记录

**逻辑**：用电量是企业生产经营的硬指标，难以造假，信用代理效果好。
