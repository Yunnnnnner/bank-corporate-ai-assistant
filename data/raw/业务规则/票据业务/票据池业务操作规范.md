# 票据池业务操作规范

**发布部门**：本行票据中心 · 公司业务部
**发布日期**：2026-01-10
**生效日期**：2026-02-01
**版本**：v2.2

---

## 第一章 总则

**第一条** 为规范本行票据池业务，依据《中华人民共和国票据法》《商业汇票承兑、贴现与再贴现管理办法》（人行金监总局令〔2022〕第4号）、《电子商业汇票业务管理办法》及本行《全面风险管理办法》制定本规范。

**第二条** **票据池业务**是指本行接受企业客户将其持有的多张未到期商业汇票质押入池，由本行按池内票据综合估值核定授信额度，向客户提供保函、信用证、贷款、承兑、贴现等融资便利的综合性票据服务。

**第三条** 票据池业务遵循"真实贸易、统一管理、综合授信、闭环操作"原则。

---

## 第二章 业务准入

**第四条** 客户准入条件：

（一）本行授信客户，信用评级 A 级及以上；
（二）有持续、稳定的票据收付业务（连续 6 个月以上）；
（三）月均池内票据余额 ≥ 1000 万元；
（四）已开立本行结算账户作为票据池主账户；
（五）已获得本行票据池业务专项授信。

**第五条** 入池票据要求：

- 仅接受**电子商业汇票**（ECDS 系统出票）
- 票据真实有效，背书连续完整
- 出票人、承兑人、收款人均为合法主体
- 不存在被止付、挂失、争议
- 票据剩余期限 ≥ 30 天（部分产品 ≥ 60 天）
- 票据未质押给其他金融机构

## 第三章 入池票据估值与折扣率

**第六条** **入池票据折扣率**（以面值为基数核定可用额度）：

### 6.1 银行承兑汇票（BAB）

| 承兑行评级 / 类型 | 折扣率 |
|---|---|
| 国有大行（工农中建交邮储） | 98% |
| 全国性股份制银行 | 95% |
| 优质城商行（一类） | 92% |
| 一般城商行（二类） | 90% |
| 农商行 | 85% |
| 村镇银行、外资行分行 | 80% |

### 6.2 商业承兑汇票（CAB）

| 承兑人评级 | 折扣率 |
|---|---|
| AAA（核心企业、央企、上市头部） | 80% |
| AA+ | 75% |
| AA | 70% |
| AA- | 60% |
| A+ | 50% |
| A | 不入池或全额保证金 |

### 6.3 财务公司承兑汇票

| 财务公司评级 | 折扣率 |
|---|---|
| 央企集团一级财务公司 | 90% |
| 大型集团财务公司 | 85% |
| 一般财务公司 | 75% |
| 小型集团财务公司 | 不入池或专项审批 |

**第七条** 折扣率动态调整：

- 承兑行/人评级下调 → 立即下调折扣率
- 票据剩余期限 < 30 天 → 折扣率下调 5 个百分点
- 大额单张 ≥ 5000万 → 由总行专项审核
- 同一承兑人入池票据集中度 > 30% → 触发预警

## 第四章 综合授信与产品组合

**第八条** 票据池可用额度 = Σ(单张票面 × 对应折扣率) − 已占用敞口

**第九条** 池内可办理的衍生产品：

| 产品 | 期限上限 | 占用比例 |
|---|---|---|
| 池内票据贴现 | 票据到期日 | 折扣后金额 |
| 票据池保函 | ≤ 1 年 | 100% 占用 |
| 票据池信用证 | ≤ 1 年 | 100% 占用 |
| 票据池流动资金贷款 | ≤ 1 年 | 100% 占用 |
| 池内票据再质押开票 | ≤ 6 个月 | 100% 占用 |
| 池内票据福费廷 | ≤ 票据到期 | 折扣后金额 |

**第十条** 操作模式：

```
企业票据流入 → 入池 → 综合授信 → 衍生产品融资 → 还款 → 票据流出
```

## 第五章 池内管理

**第十一条** 票据池实行**专户管理**：

- 票据池主账户开立专户
- 入池票据经 ECDS 质押登记
- 票据池业务系统记录所有入池/出池/解质操作

**第十二条** 池内票据状态：

- **入池**：完成质押登记
- **占用**：已用作抵押开立衍生产品
- **释放**：还款后从占用状态释放
- **出池**：到期托收 / 客户主动赎回 / 解质

**第十三条** 票据到期托收流程：

- 到期前 5 个工作日：系统自动提醒
- 到期日 T-1：将托收数据发送至承兑行
- 到期日 T：承兑行自动扣款 → 资金到账主账户
- T+1：自动归还衍生产品占用 / 余额回入企业账户

## 第六章 风险管理

**第十四条** 重点风险点：

- **承兑人违约**：商业承兑汇票到期承兑人无力付款
- **承兑行违约**：极端情况下中小银行违约
- **虚假票据**：克隆票、变造票（已基本由 ECDS 杜绝）
- **关联交易**：自开自贴、互开互贴套取信贷
- **背书断链**：背书连续性瑕疵

**第十五条** 防控措施：

- 商票必须穿透核查承兑人真实经营状况
- 自承兑商票（自开自票）原则上不入池
- 单一承兑人池内集中度限制：≤ 池总额 30%
- 关联方承兑商票入池须升级审批
- 商票到期前 15 天加强承兑人监控
- 出现承兑预警后立即降低对应折扣率或剔除出池

**第十六条** **追索权管理**：

- 商票到期承兑人未付款，本行可在合理期限内向背书人、出票人、保证人追索
- 追索时效：被拒付之日起 6 个月内
- 启动追索时立即冻结企业池内资产，必要时上报涉嫌欺诈情形

## 第七章 价格与收费

**第十七条** 票据池业务定价：

- 池内贴现：参照 DR007 + 风险加点（30~80 BP，按承兑人/出票人评级）
- 池内开立保函/信用证：参照本行保函/信用证标准费率
- 池内贷款：LPR + 加点（按授信定价规则）
- 池内承兑：手续费 0.5‰ ~ 1‰

**第十八条** 池管理费：池内余额 × 年化 0.1%~0.3%（按客户分级）
