# 票据业务操作规范
## 参照建设银行票据业务管理制度

---

## 第一章 银行承兑汇票业务

### 1.1 业务定义与产品逻辑

银行承兑汇票（BAB）是由出票人签发，委托银行在指定日期无条件支付确定金额的票据。银行在汇票上加盖承兑章后，承担到期无条件付款的责任。

**产品逻辑**
```
出票人（买方/付款方）
    │── 向银行申请承兑（缴纳保证金+授信）
    │
建行（承兑行）
    │── 审核贸易背景真实性
    │── 在票面承兑，承担第一付款责任
    │
持票人（卖方/收款方）
    │── 可持票到期托收
    │── 可向银行申请贴现，提前获得资金
    │── 可背书转让给第三方
```

### 1.2 申请承兑条件

**客户准入**
- 在银行开立结算账户，账期满6个月以上
- 与出票申请相对应的真实商品交易关系
- 纳入银行综合授信额度管理
- 无不良信用记录

**贸易背景真实性核查（三要素）**
| 要素 | 核查内容 | 常见造假手段 |
|------|---------|-----------|
| 合同 | 购销合同真实有效，签章一致 | 伪造合同、阴阳合同 |
| 发票 | 增值税发票真伪（国税系统查验） | 虚开发票 |
| 物流 | 货物运输凭证（提单/运单）或仓储证明 | 空单、循环贸易 |

**保证金缴纳标准**
| 客户评级 | 保证金比例 |
|---------|----------|
| AAA-AA | 0%（全额授信覆盖）|
| A | 20%-30% |
| BBB | 50% |
| BB及以下 | 80%-100% |

### 1.3 电子商业汇票（ECDS）操作流程

**开票流程（通过企业网银）**
```
1. 登录建行企业网银 → 票据业务 → 开出汇票
2. 填写：收款人信息、票面金额、到期日（最长6个月）
3. 上传：购销合同、增值税发票（系统自动查验）
4. 提交申请 → 银行审核（1个工作日内）
5. 审核通过 → 汇票通过ECDS系统签发
6. 收款方在ECDS确认接收
```

**贴现流程**
```
持票人申请 → 提交贴现申请（注明贴现金额/用途）
→ 银行审核（贸易背景、票据真实性）
→ 贴现利率报价
→ 借款人确认
→ 银行扣除贴息，余款转入账户
```

**贴现利率定价**
```
贴现利率 = DR007（货币市场利率基准）+ 风险溢价 + 流动性溢价
银行到期收益 = 票面金额 × 贴现利率 × 剩余天数/360
实际到账 = 票面金额 - 银行到期收益
```

### 1.4 票据池业务

**业务逻辑**
企业将持有的多张票据（包括银承和商承）汇集入池，银行综合评估票据资产质量后，给予一定比例的融资额度，支持循环借款。

**票据入池标准**
- 票据类型：电子银行承兑汇票优先，商业承兑汇票须评级达标
- 剩余期限：1个月以上
- 票面金额：单张不低于1万元
- 背书次数：不超过10次
- 无冻结、质押、挂失等权利受限

**票据池额度计算**
```
票据池额度 = Σ (各票据面值 × 对应折扣率)

折扣率标准：
- 国有大行承兑、剩余期限>3个月：98%
- 股份制银行承兑、剩余期限>3个月：95%
- 城商行承兑：90%
- 商业承兑汇票（AAA债务人）：85%
- 商业承兑汇票（AA债务人）：75%
```

### 1.5 票据风险管理

**核心风险类型**
| 风险类型 | 具体表现 | 防控措施 |
|---------|---------|---------|
| 无贸易背景 | 客户利用票据套取资金 | 严格三单核查 + 资金流向追踪 |
| 票据集中度 | 同一承兑行票据占比过高 | 单一承兑行限额管理 |
| 商承信用风险 | 承兑企业无力付款 | 债务人评级门槛 |
| 期限错配 | 贴现期限与业务不匹配 | 定期复核票据台账 |
| 操作风险 | 电票系统操作失误 | 双人复核、操作日志 |

## 第二章 商业承兑汇票

### 2.1 与银承的核心区别

| 对比维度 | 银行承兑汇票（BAB） | 商业承兑汇票（CAB） |
|---------|-------------------|-------------------|
| 承兑方 | 银行 | 付款企业自身 |
| 信用基础 | 银行信用（低风险） | 企业信用（高风险） |
| 流通性 | 高（可在银行间转贴） | 低（受债务人信用限制） |
| 适用场景 | 供应商要求安全支付 | 大型核心企业对供应商 |
| 票据贴现难度 | 容易 | 取决于开票企业信用 |

### 2.2 商承接受/贴现准入

银行受理商承贴现的承兑企业（债务人）准入标准：
- 内部评级：BBB及以上
- 近2年无失信记录
- 非房地产/城投等限制行业
- 财务健康（无持续亏损）

## 第三章 风险管理与监控

### 3.1 票据业务合规红线

**严禁行为**
1. 为无真实贸易背景的票据承兑/贴现（套票）
2. 同一笔贸易反复开票（一单多票）
3. 为关联企业之间的循环贸易提供票据融资
4. 承兑/贴现后将资金用于购买银行理财产品
5. 虚构贸易合同骗取票据融资

### 3.2 大额交易报告

- 单笔或日累计人民币200万元及以上的票据业务须向人民银行报告
- 涉嫌洗钱的票据交易须即时上报
