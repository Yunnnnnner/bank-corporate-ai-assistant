## 二、各产品线核心业务逻辑

### A. 信贷类产品业务逻辑

#### A1. 流动资金贷款

**业务定义**
向企业法人发放的，用于日常经营周转的本外币贷款。

**业务流程**
```
营销获客 → 需求确认 → 贷前调查（现场+非现场）→ 财务分析
→ 担保落实 → 客户评级 → 授信申请 → 审批委员会
→ 审批意见 → 合同签订 → 落实条件 → 受托支付/自主支付
→ 贷后检查（季度/年度）→ 到期收回/续贷
```

**核心风控节点**
| 节点 | 审查内容 | 风险信号 |
|------|---------|---------|
| 现金流 | 主营业务收入连续性 | 收入突降>30%、回款异常 |
| 负债 | 资产负债率、有息负债规模 | 隐性负债、表外担保链 |
| 还款来源 | 第一还款来源明确性 | 过度依赖资产变现 |
| 资金用途 | 专款专用、贷款划转路径 | 转入其他金融账户 |
| 关联交易 | 控股股东、实际控制人关联 | 大额关联转账 |

**定价逻辑**
- 基准：LPR（1Y）
- 加减点：= 资金成本 + 运营成本 + 风险溢价 + 目标利润
- 风险溢价 = f(评级, 担保方式, 行业系数, 区域系数)
- 最低定价不低于LPR - 30BP（内部底线）

---

#### A4. 并购贷款

**业务特点**
资金用于股权收购，风险高于一般信贷，监管有专项管理办法。

**关键约束（银保监规定）**
- 并购贷款占并购交易价款比例≤60%
- 贷款期限≤7年
- 并购方应提供≥并购价款20%的自有资金
- 禁止用于：无实际并购标的的概念并购、杠杆收购中目标企业资产负债率>70%的

**尽职调查重点**
1. 标的企业估值合理性
2. 交割条件与时间线
3. 收购后整合计划
4. 还款来源（目标公司经营现金流+分红）

---

#### A5. 银团贷款

**牵头行职责**
- 组建银团、谈判条款
- 起草信息备忘录（IM）
- 承销未认购份额的风险

**参团行职责**
- 独立审批，不依赖牵头行结论
- 签署银团协议、参与代理行安排

**代理行职责**
- 贷款发放与回收（资金代理）
- 管理费分配
- 贷后事项通知

### B. 票据业务逻辑

#### B1. 银行承兑汇票完整流程

```
出票人申请开票
    │
    ├── 授信额度核查（票据专项额度 or 综合授信敞口）
    ├── 保证金收取（0%-100%，按风险等级）
    ├── 银行审核（贸易背景真实性）
    └── 出具承兑汇票（电票通过ECDS系统）
           │
           ↓ 持票人可选择：
    ┌──────┼──────┐
    │      │      │
    贴现  背书  到期托收
    │      │
   贴现行 下家持票人
    │
 转贴现（银行间）
    │
 再贴现（央行）
```

**票据风控要点**
- 贸易背景真实性核查（合同+发票+物流三单一致）
- 禁止无贸易背景开票（套利套现严打）
- 关注客户票据敞口集中度
- 电票优先（纸票风险高：伪造、克隆）

#### B4. 票据池业务逻辑

企业将多张票据入池，银行综合评估票据资产质量，给予一定比例的循环融资额度。

**核价公式**
```
票据池融资额度 = Σ(票据面值 × 折扣率)
折扣率 = f(承兑行评级, 剩余期限, 是否电票)
```

### C. 供应链金融业务逻辑

#### 三种模式对比

| 模式 | 融资主体 | 信用基础 | 典型产品 | 还款来源 |
|------|---------|---------|---------|---------|
| 应付账款模式 | 上游供应商 | 核心企业确认应付 | e点通 | 核心企业付款 |
| 买方融资模式 | 下游经销商 | 核心企业担保/增信 | e销通 | 销售回款 |
| 存货融资模式 | 借款人 | 存货质押 | 仓单质押 | 货物变现 |

#### 应收账款保理逻辑

```
应收账款转让
    │
    ├── 有追索保理：债务人违约，保理商向转让方追偿
    └── 无追索保理：信用风险完全由保理商承担
           │
           ↓ 风险定价更高，要求债务人信用优质
```

**保理资产准入标准**
- 应收账款真实、合法、无权利瑕疵
- 债务人信用评级 BBB 及以上
- 账期不超过 180 天
- 非关联交易（防止虚假贸易）

### D. 贸易融资业务逻辑

#### D1. 信用证全流程

```
买方(进口商)    建设银行(开证行)    卖方(出口商)    出口行(通知行/议付行)
     │                │                  │                   │
     ├─申请开证──────→│                  │                   │
     │                ├─开立信用证──────────────────────────→│
     │                │                  │←─通知信用证────────┤
     │                │                  ├─发货─────────────→│
     │                │                  ├─提交单据─────────→│
     │                │←─寄单索汇────────────────────────────┤
     │                ├─审单（5个工作日）│                   │
     │                ├─付款/拒付────────────────────────────→│
     ├─通知到单────────┤                  │                   │
     ├─付款赎单────────→                  │                   │
```

**单据审核"严格相符"原则**
- 受益人名称与信用证一致
- 金额不超证面金额
- 装运日期、效期均在要求范围
- 商业发票、提单、装箱单三单一致

### E. ABS/债券投行业务逻辑

#### E2. 信贷资产证券化（CLO）完整流程

```
【离线准备阶段】(3-6个月)
1. 发起行确定证券化目标（出表、融资、监管资本节约）
2. 基础资产池筛选
   - 剔除：逾期>90天、诉讼中、重组贷款
   - 入池标准：按监管要求设定
3. 聘请中介机构（律师/会计师/评级机构/主承销商）
4. 设立信托计划（受托机构：银行/信托公司）
5. 交易结构设计（优先/次级分层）
6. 信用增级方案（内部超额抵押+外部担保）
7. 信用评级（AAA/AA/A各档）

【发行阶段】(1-2个月)
8. 监管报批（人行/银保监备案或报批）
9. 路演推介（机构投资者）
10. 簿记建档、定价
11. 证券发行、交割
12. 资金回流发起行

【存续期管理阶段】(贷款剩余期限)
13. 服务机构月度报告
14. 优先级本息按期支付
15. 早偿/违约事件处理
16. 到期清算
```

**分层结构示例（CLO）**
```
优先A级（AAA）：占70%，利率最低，优先受偿
优先B级（AA） ：占15%，利率中等
次级（无评级）：占15%，由发起行持有（5%以上留存）
超额利差账户：用于覆盖早期损失
```

#### E4. 绿色债券/碳中和债

**认定标准**
- 募集资金100%用于绿色项目（清洁能源/节能减排/生态保护）
- 须提供第三方绿色认证报告
- 每半年披露资金使用情况
